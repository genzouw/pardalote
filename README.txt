$1. 認証
    #1-1) 各種設定
        * メニューの"設定"から設定できます。

        * 以下の設定が可能です。
            - facebook認証
                facebookへの認証手続きを行います。
                ブラウザが起動し認証手続きを開始します。
            - facebook認証解除
                facebookへの認証解除手続きを行います。
                一度設定したユーザーを解除し、別ユーザーを設定する際にご利用下さい。



$2. 記事管理機能 タブ で提供している機能
    #2-1) 新規登録
        * 上部に入力した情報を登録できます。

        * 以下の情報が入力できます。
            項目名          入力は必須か？
            ----------------------------------------
            件名            ○
            本文            ○
            フォト
            送信時間指定

    #2-2) 登録データ一覧表示
        * 新規登録・変更した情報が一覧で表示されます。

    #2-3) 変更/削除

        * "登録データ一覧表示"内のデータをダブルクリックすると、
          選択された情報が上部に表示されます。

        * 以下の情報が入力できます。
            項目名          入力は必須か？
            ----------------------------------------
            件名            ○
            本文            ○
            フォト
            送信時間指定

        * 既に送信済みの情報は一覧でダブルクリックしても変更できません。


    #2-4) 各種設定
        * メニューの"設定(記事)"から設定できます。

        * 以下の設定が可能です。
            - 自動送信ON
                下部に表示されている一覧情報のうち、
                未送信（送信日時が空）情報を定期的に送信します。
            - 送信後に送信データを削除する
                "自動送信ON" の場合、
                送信済データを保存せず削除します。

$5. いいね！管理機能 タブ で提供している機能
    #5-1) 各種設定
        * メニューの"設定(いいね！)"から設定できます。

        * 以下の設定が可能です。
            - 自動クリックON
                全ての友人のウォールに対して、
                いいね！のクリックを自動的に行います。
                いいね！クリックから次のいいね！クリックまでの間隔は
                10～300秒の間でランダムに決定されます。
